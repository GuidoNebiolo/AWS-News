# AWS News 2024-09-12

## Summary

- [Amazon RDS for MySQL now offers zero-ETL integration with Amazon Redshift, enabling near real-time analytics and ML on petabytes of transactional data without complex data pipelines.](https://aws.amazon.com/about-aws/whats-new/2024/09/amazon-rds-mysql-zero-etl-integration-redshift-generally-available)

- [Amazon Redshift Serverless is now available in the AWS Asia Pacific (Hong Kong) and Israel (Tel Aviv) Regions, allowing users to run and scale analytics without managing data warehouse clusters.](https://aws.amazon.com/about-aws/whats-new/2024/09/amazon-redshift-serverless-aws-asia-pacific-hong-kong-israel-tel-aviv-regions)

- [Amazon SageMaker Inference introduces sticky session routing, improving performance and user experience for generative AI applications by leveraging previously processed information.](https://aws.amazon.com/about-aws/whats-new/2024/09/sticky-session-routing-amazon-sagemaker-inference)

- [AWS Network Firewall now supports AWS PrivateLink, enabling private access and management of Network Firewalls without using the public internet.](https://aws.amazon.com/about-aws/whats-new/2024/09/aws-network-firewall-privatelink)

- [Amazon EC2 R6in and R6idn instances are now available in the Asia Pacific (Sydney) region, offering increased network bandwidth and performance for memory-intensive workloads.](https://aws.amazon.com/about-aws/whats-new/2024/09/amazon-ec2-r6in-r6idn-instances-additional-region)

- [AWS Application Migration Service now supports Trend Micro post-launch action, allowing automatic installation of the Trend Micro Vision One Server & Workload Protection Agent on migrated instances.](https://aws.amazon.com/about-aws/whats-new/2024/09/aws-application-migration-service-launch-action/)

- [AWS Elemental Media Services now support live AV1 encoding, enabling lower bitrate streaming with similar viewing experience and reduced CDN costs.](https://aws.amazon.com/about-aws/whats-new/2024/09/aws-elemental-media-services-live-av1-encoding)

- [AWS Elemental MediaLive Anywhere is now generally available, allowing customers to run live video transcoding on their on-premises hardware with cloud control and pay-as-you-go pricing.](https://aws.amazon.com/about-aws/whats-new/2024/09/aws-elemental-medialive-anywhere-video-encoding-hardware/)

- [AWS Backup Audit Manager adds a new control to audit resources inside logically air-gapped vaults, helping customers meet compliance requirements for data protection.](https://aws.amazon.com/about-aws/whats-new/2024/09/aws-backup-audit-manager-control-audit-resources/)

- [AWS Glue Data Catalog now supports storage optimization of Apache Iceberg tables, automatically removing unnecessary data files to reduce costs and improve query performance.](https://aws.amazon.com/about-aws/whats-new/2024/09/aws-glue-data-catalog-optimization-apache-iceberg-tables/)

- [Amazon ECR now supports dual-layer server-side encryption in the AWS GovCloud (US) Regions, meeting stronger compliance and regulatory requirements for highly regulated environments.](https://aws.amazon.com/about-aws/whats-new/2024/09/amazon-ecr-dual-layer-server-side-encryption-govcloud/)

## Commentary

These announcements demonstrate AWS's commitment to continually enhancing its services and expanding their availability. The new features and improvements span various areas, including analytics, machine learning, networking, migration, media services, and data protection. These advancements aim to provide customers with more efficient, cost-effective, and secure solutions for their cloud computing needs. By focusing on performance optimization, compliance, and ease of use, AWS is empowering businesses to innovate faster and operate more effectively in the cloud.
