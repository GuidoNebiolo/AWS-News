# AWS News 2024-09-09

## Summary

- [AWS Elastic Beanstalk now supports IPv6 inbound traffic, enabling dual-stack functionality for improved network compatibility and flexibility.](https://aws.amazon.com/about-aws/whats-new/2024/09/aws-elastic-beanstalk-ipv6-inbound-traffic-service-endpoints)
- [Amazon Aurora expands R7g Graviton3-based instance family availability to 15 additional regions, offering improved performance and cost-efficiency.](https://aws.amazon.com/about-aws/whats-new/2024/09/amazon-aurora-r7g-graviton3-based-instance-family-additional-regions)
- [AWS IoT SiteWise Edge now supports secondary sensors through CloudRail integration, enabling data collection from over 12,000 sensors and brownfield equipment.](https://aws.amazon.com/about-aws/whats-new/2024/09/secondary-sensor-aws-iot-sitewise-edge-cloudrail)
- [Amazon IVS Real-Time Streaming adds support for RTMP ingest, enhancing compatibility with a wide range of software and hardware encoders.](https://aws.amazon.com/about-aws/whats-new/2024/09/amazon-ivs-real-time-streaming-rtmp-ingest/)
- [AWS IoT SiteWise Edge now supports 100+ industrial protocols through Litmus Edge integration, simplifying data ingestion from various industrial equipment.](https://aws.amazon.com/about-aws/whats-new/2024/09/aws-iot-sitewise-edge-100-protocols-litmus-edge/)
- [AWS Resource Access Manager now supports AWS PrivateLink, enabling secure resource sharing within Amazon VPCs without public internet exposure.](https://aws.amazon.com/about-aws/whats-new/2024/09/aws-resource-access-manager-privatelink)
- [AWS Elemental MediaConnect introduces input thumbnail images for improved monitoring and visual feedback of live content sources.](https://aws.amazon.com/about-aws/whats-new/2024/09/aws-elemental-mediaconnect-input-thumbnail-images/)
