# AWS News 2024-09-13

## Summary

- [Amazon Connect expands AWS CloudFormation support for agent hierarchies and status, enabling programmatic deployment and efficient management of contact center configurations.](https://aws.amazon.com/about-aws/whats-new/2024/09/amazon-connect-aws-cloudformation-agent-hierarchies)
- [AWS WAF Bot Control Managed Rule enhances bot detection capabilities with new features like token reuse detection and expanded bot categories.](https://aws.amazon.com/about-aws/whats-new/2024/09/aws-waf-bot-control-managed-rule-bot-detection-capabilities)
- [Amazon QuickSight now supports Direct Query for Google BigQuery connector, allowing near real-time querying without data size limitations.](https://aws.amazon.com/about-aws/whats-new/2024/09/amazon-quicksight-direct-query-functionality-google-bigquery-connector)
- [Amazon Redshift Serverless increases maximum base capacity to 1024 Redshift Processing Units, providing greater flexibility for complex workloads.](https://aws.amazon.com/about-aws/whats-new/2024/09/amazon-redshift-serverless-capacity-1024-processing-units/)
- [Amazon RDS for SQL Server now supports minor versions 2019 CU28 and 2022 CU14, offering performance enhancements and security fixes.](https://aws.amazon.com/about-aws/whats-new/2024/09/amazon-rds-sql-server-minor-versions-cu28-cu14/)
- [Amazon Cognito user pools now offer email as a multi-factor authentication (MFA) option, expanding security choices for web and mobile apps.](https://aws.amazon.com/about-aws/whats-new/2024/09/amazon-cognito-user-pools-email-authentication-option/)
- [Amazon Bedrock Knowledge Bases introduces cross-region inference support, enhancing throughput and resilience during peak demand periods.](https://aws.amazon.com/about-aws/whats-new/2024/09/amazon-bedrock-knowledge-bases-cross-region-inference)
- [Amazon Neptune Database expands availability to Asia Pacific (Jakarta) and Europe (Spain) Regions, offering fully managed graph database services in more locations.](https://aws.amazon.com/about-aws/whats-new/2024/09/amazon-neptune-database-asia-pacific-jakarta-region)
