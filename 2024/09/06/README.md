# AWS News 2024-09-06

## Summary

- [Amazon Kinesis Data Streams now supports FIPS 140-3 enabled interface VPC endpoints, enhancing security for regulated workloads and enabling compliance with US federal government requirements.](https://aws.amazon.com/about-aws/whats-new/2024/09/amazon-kinesis-data-streams-fips-140-3-interface-vpc-endpoint)
- [Amazon RDS Custom for SQL Server expands to three additional AWS Regions, offering more flexibility and managed database features for customers using licensed SQL Server media.](https://aws.amazon.com/about-aws/whats-new/2024/09/amazon-rds-custom-sql-server-additional-aws-regions)
- [PostgreSQL 17 RC1 is now available in the Amazon RDS Database Preview Environment, allowing customers to evaluate new features and improvements before general release.](https://aws.amazon.com/about-aws/whats-new/2024/09/postgresql-17-rc1-amazon-rds-database-preview-environment)
- [CloudWatch Application Signals introduces request-based Service Level Objectives (SLOs), enabling more precise tracking of service quality and performance.](https://aws.amazon.com/about-aws/whats-new/2024/09/cloudwatch-application-signals-service-level-objectives/)
- [Amazon ECS now supports AWS Graviton-based Spot compute with AWS Fargate, offering cost-effective and efficient serverless container deployments.](https://aws.amazon.com/about-aws/whats-new/2024/09/amazon-ecs-graviton-based-spot-compute-fargate/)
- [Amazon Timestream for InfluxDB expands to Canada, London, and Paris AWS regions, providing fully managed time-series database capabilities to more customers.](https://aws.amazon.com/about-aws/whats-new/2024/09/amazon-timestream-influxdb-canada-london-paris-regions/)
- [Amazon RDS for MariaDB now supports minor versions 10.11.9, 10.6.19, and 10.5.26, offering improved security, bug fixes, and performance enhancements.](https://aws.amazon.com/about-aws/whats-new/2024/09/amazon-rds-mariadb-minors-10/)
