# AWS News 2024-08-30

## Summary

- [AWS Control Tower now allows Organizational Units to contain up to 1,000 accounts, enabling greater flexibility and scalability for implementing governance best practices across larger organizations.](https://aws.amazon.com/about-aws/whats-new/2024/08/organizational-units-aws-control-tower-1000-accounts)

- [Amazon Redshift Serverless now supports AWS PrivateLink, enhancing security by allowing direct connections within the AWS network without the need for public internet exposure.](https://aws.amazon.com/about-aws/whats-new/2024/08/amazon-redshift-serverless-aws-privatelink)

- [AWS IoT SiteWise introduces versioning for asset and component models, helping industrial customers manage model evolution more effectively and recover from failed states.](https://aws.amazon.com/about-aws/whats-new/2024/08/aws-iot-sitewise-models-support-versioning)

- [AWS Security Hub launches 8 new security controls, expanding support for additional AWS services and enhancing security posture monitoring capabilities.](https://aws.amazon.com/about-aws/whats-new/2024/08/aws-security-hub-8-new-security-controls/)

- [AWS Config conformance packs and organization-level management capabilities are now available in 12 additional AWS Regions, simplifying compliance management across AWS Organizations.](https://aws.amazon.com/about-aws/whats-new/2024/08/aws-config-conformance-packs-additional-regions)

- [AWS Backup extends support for Cross-Region backup of Amazon Neptune to additional regions, improving data resiliency and disaster recovery capabilities.](https://aws.amazon.com/about-aws/whats-new/2024/08/aws-backup-support-cross-region-backup-amazon-neptune)

- [Amazon EMR Managed Scaling is now Application Master placement aware, improving application resiliency and cluster scaling based on executor and ApplicationMasters demand.](https://aws.amazon.com/about-aws/whats-new/2024/08/amazon-emr-managed-scaling-application-master-placement-aware)

- [Amazon Personalize enhances automatic solution training, allowing developers to modify training configurations after solution creation for greater flexibility and adaptability.](https://aws.amazon.com/about-aws/whats-new/2024/08/amazon-personalize-automatic-solution-training/)

- [AWS WAF now supports lower rate limit thresholds for rate-based rules, enabling more granular control over traffic and improved mitigation of sudden usage increases or malicious activity.](https://aws.amazon.com/about-aws/whats-new/2024/08/aws-waf-rate-based-rules-lower-rate-limits/)

- [AWS Amplify introduces scheduled cron jobs and streaming logs for its Functions capability, enhancing automation and real-time debugging for serverless applications.](https://aws.amazon.com/about-aws/whats-new/2024/08/aws-amplify-function-cron-jobs-streaming-logs/)

## Commentary

These announcements demonstrate AWS's commitment to continuously improving and expanding its services to meet the evolving needs of customers. The enhancements across various AWS offerings provide greater flexibility, security, and scalability, enabling businesses to optimize their cloud infrastructure and operations. By focusing on areas such as governance, data protection, industrial IoT, and serverless computing, AWS is empowering organizations to innovate faster and more efficiently while maintaining robust security and compliance standards. These updates reflect AWS's dedication to delivering cutting-edge cloud solutions that drive business value and support customers in their digital transformation journeys.
