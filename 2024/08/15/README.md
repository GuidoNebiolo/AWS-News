# AWS News 2024-08-15

## Summary

- [Amazon VPC Lattice expands to 7 new regions, enhancing service-to-service communication and connectivity options for customers across 18 AWS regions.](https://aws.amazon.com/about-aws/whats-new/2024/08/amazon-vpc-lattice-additional-regions)

- [AWS Support launches an official channel on AWS re:Post, providing customers with curated content and expert insights for operating at scale on AWS.](https://aws.amazon.com/about-aws/whats-new/2024/08/aws-support-official-channel-re-post)

- [AWS Control Tower introduces landing zone version selection, offering customers more flexibility in managing their multi-account AWS environments.](https://aws.amazon.com/about-aws/whats-new/2024/08/aws-control-tower-landing-zone-version-selection)

- [Amazon ECS now allows container restarts without full task relaunches, improving container resiliency and reducing recovery time for transient failures.](https://aws.amazon.com/about-aws/whats-new/2024/08/amazon-ecs-restart-containers-task-relaunch/)

- [AWS CodeBuild introduces support for GitHub Apps and multiple access tokens via AWS Secrets Manager, enhancing security and flexibility for source repository access.](https://aws.amazon.com/about-aws/whats-new/2024/08/aws-codebuild-github-apps-access-source-repositories)

- [Amazon QuickSight extends Customer Managed Key Encryption (CMK) to data source information, providing enhanced security for customers' sensitive credentials.](https://aws.amazon.com/about-aws/whats-new/2024/08/amazon-quicksight-cmk-data-source-information)

- [AWS Ground Station launches digital twin feature, enabling faster integration and testing of satellite communication workflows in a virtual environment.](https://aws.amazon.com/about-aws/whats-new/2024/08/digital-twin-available-aws-ground-station/)

- [Amazon EC2 G6e instances powered by NVIDIA L40S Tensor Core GPUs are now generally available, offering improved performance for machine learning and spatial computing workloads.](https://aws.amazon.com/about-aws/whats-new/2024/08/amazon-ec2-g6e-instances/)
