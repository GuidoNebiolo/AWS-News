# AWS News 2024-07-03

## Summary

- [Amazon RDS Snapshot Export to S3 now available in eight additional AWS regions, enabling easier data extraction for analytics and machine learning purposes.](https://aws.amazon.com/about-aws/whats-new/2024/07/amazon-rds-snapshot-export-s3-additional-aws-regions)

- [AWS Launch Wizard introduces programmatic deployments through APIs and CloudFormation templates, offering more flexibility in deploying third-party applications.](https://aws.amazon.com/about-aws/whats-new/2024/07/aws-launch-wizard-deployments-apis-cloudformation-templates/)

- [Amazon DataZone introduces fine-grained access control, allowing data owners to restrict access at row and column levels for enhanced data governance.](https://aws.amazon.com/about-aws/whats-new/2024/07/amazon-datazone-fine-grained-access-control/)

- [Amazon RDS now supports M6gd and R6gd database instances in AWS GovCloud (US) Regions, expanding options for government and regulated industry customers.](https://aws.amazon.com/about-aws/whats-new/2024/07/amazon-rds-m6gd-r6gd-database-instances-govcloud/)

- [AWS Lambda adds support for runtime management controls in AWS GovCloud (US) Regions, providing more visibility and control over runtime updates.](https://aws.amazon.com/about-aws/whats-new/2024/07/aws-lambda-runtime-management-controls-govcloud-regions)

- [Amazon Q Developer is now generally available in the Visual Studio IDE, offering AI-powered assistance for software development tasks.](https://aws.amazon.com/about-aws/whats-new/2024/07/amazon-q-developer-ga-visual-studio-ide)

- [AWS Lambda introduces advanced logging controls, enabling easier search, filtering, and aggregation of function logs.](https://aws.amazon.com/about-aws/whats-new/2024/07/aws-lambda-search-filter-aggregate-function-logs)
