# AWS News 2024-07-30

## Summary

- [AWS Graviton-based EC2 instances now support hibernation, enabling customers to pause and resume instances at scale, lowering costs and achieving faster startup times](https://aws.amazon.com/about-aws/whats-new/2024/07/aws-graviton-based-ec2-instances-hibernation)
- [Amazon Connect expands support for Inbound Direct Dial (DID) telephone numbers in Vietnam, enhancing global communication capabilities](https://aws.amazon.com/about-aws/whats-new/2024/07/amazon-connect-inbound-did-calling-vietnam)
- [Amazon Neptune Analytics introduces smaller capacity units (32 and 64 m-NCUs), providing greater flexibility and cost efficiency for smaller graph sizes](https://aws.amazon.com/about-aws/whats-new/2024/07/amazon-neptune-analytics-smaller-capacity-units/)
- [AWS Account creation now supports bank account integration in Germany, offering more payment options for customers](https://aws.amazon.com/about-aws/whats-new/2024/07/aws-account-using-bank-account-germany)
- [AWS Elastic Disaster Recovery introduces Flexible Instance Types, reducing the likelihood of failed recovery attempts due to capacity shortages](https://aws.amazon.com/about-aws/whats-new/2024/07/aws-elastic-disaster-recovery-flexible-instance-types-launch-setting)
- [AWS IoT SiteWise Edge on Siemens Industrial Edge is now generally available, bridging the gap between OT and IT for industrial customers](https://aws.amazon.com/about-aws/whats-new/2024/07/aws-iot-sitewise-edge-siemens-industrial-edge/)
- [Amazon AppStream 2.0 now supports Red Hat Enterprise Linux application and desktop streaming, expanding operating system options for customers](https://aws.amazon.com/about-aws/whats-new/2024/07/amazon-appstream-red-hat-linux-desktop-streaming/)

## Commentary

These announcements demonstrate AWS's commitment to enhancing its services across various domains, from compute and analytics to disaster recovery and industrial IoT. By introducing new features and expanding support for existing services, AWS continues to empower customers with more flexibility, cost-efficiency, and improved performance. These innovations enable businesses to leverage cloud technologies more effectively, driving digital transformation and unlocking new possibilities for growth and innovation.
