# AWS News 2024-07-01

## Summary

- [AWS Direct Connect now offers native 400 Gbps Dedicated Connections at select locations, enabling higher bandwidth connectivity for large-scale data transfer applications.](https://aws.amazon.com/about-aws/whats-new/2024/07/aws-direct-connect-native-400-gbps-dedicated-connections-select-locations)
- [Amazon OpenSearch Ingestion adds support for ingesting data from self-managed sources, simplifying data migration and replication from OpenSearch, Elasticsearch, and Apache Kafka clusters.](https://aws.amazon.com/about-aws/whats-new/2024/07/amazon-opensearch-ingesting-data-self-managed-sources)
- [AWS announces a streamlined Migration Acceleration Program (MAP) funding and approval process in AWS Partner Central, supporting partner-led migrations up to $10M in annual recurring revenue.](https://aws.amazon.com/about-aws/whats-new/2024/07/streamlined-map-funding-approval-aws-partner-central)
- [Amazon RDS for PostgreSQL now supports additional PL/Rust crates, including serde, serde_json, regex, and url, enabling more advanced extension development.](https://aws.amazon.com/about-aws/whats-new/2024/07/rds-postgresql-pl-rust-crates-serde-json-regex-url/)
- [Amazon S3 Access Grants now integrate with open source Python frameworks, simplifying data permissions management in various Python-based applications.](https://aws.amazon.com/about-aws/whats-new/2024/07/amazon-s3-access-grants-integrate-open-source-python/)
- [Amazon Connect introduces the ability to preferentially route contacts to specific agents within a queue, enhancing personalized customer service.](https://aws.amazon.com/about-aws/whats-new/2024/07/amazon-connect-preferentially-route-contacts-agents-queue)
- [Amazon S3 Access Grants now integrate with Amazon SageMaker Studio, streamlining access to ML datasets for model training and inference.](https://aws.amazon.com/about-aws/whats-new/2024/07/amazon-s3-access-grants-integrate-sagemaker-studio/)
- [Amazon API Gateway WebSocket APIs are now available in 7 additional AWS Regions, enabling real-time bi-directional communication across all commercial AWS Regions.](https://aws.amazon.com/about-aws/whats-new/2024/07/amazon-api-gateway-websocket-apis-additional-aws-regions)
- [AWS Application Migration Service now supports Dynatrace post-launch action, automating the installation of Dynatrace agent on migrated instances for enhanced observability.](https://aws.amazon.com/about-aws/whats-new/2024/07/aws-application-migration-service-dynatrace-post-launch-action)

## Commentary

These announcements demonstrate AWS's commitment to continuously improving and expanding its services to meet customer needs. By enhancing connectivity, simplifying data management, and expanding regional availability, AWS is empowering customers to build more efficient, scalable, and innovative solutions. The focus on partner support and integration with popular tools further strengthens the AWS ecosystem, providing customers with a rich set of options to accelerate their cloud journey and drive digital transformation.
